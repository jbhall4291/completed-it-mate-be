Backend API powering [Completed It Mate](https://completeditmate.app) game tracker, built with **Node.js + Express**, **TDD via Jest + SuperTest**, and a **MongoDB Atlas** data store.

Fully tested via **GitHub Actions CI** with automated coverage reporting.

## Responsibilities

- REST API for user libraries, game status updates, and community statistics
- Weekly scheduled ingestion job (cron) to add new releases/upcert existing games
- Manual seed and maintenance scripts used during development and iteration
